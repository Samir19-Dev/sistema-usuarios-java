# Sistema de Usuarios en Java 👨‍💻

Aplicación desarrollada en Java que permite gestionar usuarios mediante operaciones básicas (CRUD).

## 🚀 Funcionalidades
- Crear usuarios
- Listar usuarios
- Eliminar usuarios
- Interfaz por consola

## 🛠️ Tecnologías utilizadas
- Java
- Programación orientada a objetos

## ▶️ Cómo ejecutar
1. Abrir el proyecto en tu IDE (NetBeans, Eclipse o IntelliJ)
2. Ejecutar la clase principal `Main.java`
3. Usar el menú en consola
